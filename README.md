# Telegram Bot для удаления сообщений о входе/выходе

Бот автоматически удаляет сообщения о том, что кто-то зашел или вышел из канала/группы.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

3. Добавьте токен бота в файл `.env`:
```
BOT_TOKEN=ваш_токен_от_BotFather
```

## Получение токена бота

1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен в файл `.env`

## Запуск

```bash
python bot.py
```

## Настройка прав бота

Для работы бота необходимо:

1. Добавить бота в канал/группу как администратора
2. Дать боту права на удаление сообщений:
   - В настройках группы/канала → Администраторы → выберите бота
   - Включите право "Удалять сообщения"

## Как это работает

Бот отслеживает все сообщения в каналах/группах, где он является администратором. Когда приходит системное сообщение о входе или выходе пользователя, бот автоматически его удаляет.
